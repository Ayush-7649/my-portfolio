SCT_CS_1 – Caesar Cipher Encryption Tool

Description: A Python program that encrypts and decrypts messages using the classical Caesar Cipher algorithm. Users can input a message and shift value to encode or decode text.

Tech Stack: Python

  • Supports both encryption and decryption
  • Works with uppercase and lowercase letters
  • User-defined shift key and terminal-based interaction
  • Demonstrates classical cryptography principles

SCT_CS_2 – Image Encryption via Pixel Manipulation

Description: A basic image encryption tool using pixel-level operations like XOR and row swapping. It encrypts and decrypts images using a numeric key.

Tech Stack: Python, Pillow, NumPy

  • Encrypts PNG/JPG images with a numeric key
  • Decrypts using the same key for restoration
  • Works with RGB and grayscale formats
  • Educational tool for understanding image security

SCT_CS_3 – Password Strength Checker

Description: A Python tool that evaluates password strength based on length, character variety, and symbol usage. Offers feedback and suggestions to improve security.

Tech Stack: Python (using re module)

  • Checks for uppercase, lowercase, digits, and special characters
  • Classifies passwords as Weak, Medium, or Strong
  • No external libraries required
  • Useful for login systems and form validation

SCT_CS_4 – Keylogger Program

Description: A basic keylogger developed for educational purposes. It records keystrokes and logs them to a file while running locally.

Tech Stack: Python, pynput library

  • Captures regular and special key inputs
  • Logs keystrokes to keystrokes.log
  • Can be stopped safely with the ESC key
  • Demonstrates ethical cybersecurity practices
